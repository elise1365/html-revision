Uniform resource locators

Two types of URLs

Absolute URLs include the scheme - "https://bbc.co.uk/news"

Relative URLs refer to resources on the same server - "sections/contact.html" OR ".../index.html"

URL uses 2 know

  • Target attribute opens a link in a new tab/window - a href="example.html" target="_blank"
  • Fragment can link to a specific part of a page - a href="example.html#ch2"
  • mailto can be used to launch a users email program to send an email - a href="mailto:up2109470@myport.ac.uk"
  • tel provides a link to the dialer - a href="tel:+4400000"
