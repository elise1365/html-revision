Basics

Scalable Vector Graphics

SVG is smaller than bitmap graphics

Better than bitmapped graphics because SVGs scale better than pixels, which become large and blocky

Types of sizing

  1. Explicit sizing is where you specify the exact width and height of an SVG element
  2. Responsive sizing is where the element is scaled dynamiclly based on the size of the parent container/viewport

CSS

Can always put fill, stroke etc of a shape in CSS

Path

To create lines/ more complex shapes use path - path d=""

  • M - moves pen tip absolute
  • m - moves pen top relative to last point

m 10 20

To draw use:

  • L/l
  • C - cubic bezier
  • Q - quadratic bezier

Attributes 2 know

  • stroke_width="10"
  • fill="none"

Elements

Circles

Created using the circle element

Takes 3 parameters

  1. cx
  2. cy
  3. r

Ellipses

Like circles but have 2 radii - cx, cy, rx, ry

Lines

Draws a line between 2 points, takes 4 parameters - x1, y1, x2, y2

Polylines

Connects multiple points together, each point is written as a pair of coords and separated by a space

polyline points="20,20 20,80 80,80"

Polygons

Same as a polyline but automatically closes the path

Text

Must have an x and y point

text x="20" y="20"

Grouping

g element groups elements together - groups are useful because they reduce the need for repeated content

Defs and use

These elements allow for a group of elements to be defined and used in different places throughout an image

Transform

Can alter an element:

  • Translate
  • Scale
  • Rotate
  • Skew x/y
  • Matrix
