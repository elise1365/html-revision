CSS

Cascading style sheets

You define rules for your html document

Selectors

  • p/article/section etc
  • * - all
  • section, p/section p - multiple selectors
  • #section - id selector, to link this do p id="section"
  • .myclass - class selector, to link this do p class="myclass"

To link a CSS to a HTML docuent do:

link rel="stylesheet" title="mystyle" href="mystyle.css"

This goes in the head after the title
